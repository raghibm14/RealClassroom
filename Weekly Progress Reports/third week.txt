Developing the classroom.
Using:
i) Pro Builder for developing the classroom structure
ii) Mezanix textures for adding textures to the classroom walls
iii) TL_Grass material to add grass texture to the terrain outside the classroom
iv) Lamp models from internet for adding lamp 3d model in the classroom, adding point lights to them.
v) A ceiling light is to be imported and baked light as well as spotlight has to be added to it with pixel count decreased to 2 otherwise smartphones won't be able to run it smooth.
vi) Bench models have been added and some posters have to be added too.
